## Slide 6：1.3 创意方案 — 核心功能详解（Skill 1-2）

### Skill 1：AI 找款（爆款雷达）

**三层处理架构：自研聚类引擎 + 趋势过滤引擎 + GATv2**

| 层级 | 来源 | 功能 |
|------|------|------|
| L1 风格聚类 | 自研聚类引擎 | FAISS 向量检索（OpenCLIP 嵌入），按视觉/情感相似度聚类 |
| L2 趋势判断 | 趋势过滤引擎 | 区分真实趋势 vs 短暂噪音 |
| L3 爆款预测 | 我们独有 | GATv2 + LightGBM 对候选款打 S/A/B |

**产品界面（四标签页工作台）：**

| 标签页 | 功能 |
|--------|------|
| 流程工作台 | 三步引导：对齐风格簇 → 参考爆款 → 货源确认 |
| 市场与簇 | 类目全局数据 + 簇卡片排序（S/A/B + 簇均销 / 视觉相似优先）+ 簇内画廊 |
| 测款与复盘 | 延伸方向 A/B/C + 测款 SOP + 风险评估 |
| 延伸设计 | **浏览器端 ONNX 前景分割 → 换色/换图案预览 → GPT-image-1 AI 生图** |

**延伸设计引擎详解：**
- 前景分割：浏览器端 ONNX 模型，实时提取服装主体
- 换色预览：Canvas 蒙版叠加，5 种预设色 + 4 种图案
- 两种改款模式：
  - 提示词改款：自然语言描述 → AI 生图
  - 设计点库：元素库/面料库/廓形库选择 → 组合生成精准 prompt → img2img
- 真实调用 GPT-image-1 / GPT-image-1-mini 模型生成图片
- 基于簇语义标签（semantic.groups）自动生成更准的编辑 prompt

**数据支撑：**
- 支持 CSV 离线数据上传 + 实时聚类（HDBSCAN / KMeans）
- 可选 GPT-4o 语义标注（45 标签分类体系）
- OpenCLIP 向量嵌入 API

### Skill 1.5：测款验证

**从"凭感觉"到"数据驱动"的测款 SOP：**
- 预测销量区间（P25/P50/P75）基于簇历史数据，附 S/A/B 等级
- 8 维度测款数据录入（上架天数、日均花费、CTR、CVR、加购率、收藏率、件数、成本价）
- **小模型回调 + LLM 叙事融合**：
  - 小模型基于实测数据调整预测区间
  - LLM（qwen-max）读取选品等级 + 原区间 + 测款结果，生成诊断叙事
  - 融合输出统一判断句
- 综合评分表盘（0-100）→ "建议放量" 或 "建议换款"
- 维度对比条：CTR/CVR/加购率/ROI vs 基准线
- 放量预测：月营收、月利润、回本天数
- 测款数据匿名提交到簇级统计池，反馈模型

## Slide 7：1.3 创意方案 — 核心功能详解（Skill 3-4）

### Skill 3：上架优化（一键生成上架包 + AI 改图）

**四标签页工作台：**

| 标签页 | 功能 |
|--------|------|
| 待办清单 | 进度仪表盘（环形图 + 5 分项进度条）+ AI 建议优先操作（P0/P1 + 跨标签跳转） |
| 标题·详情·SKU | 3 版标题 + 详情文案 + SKU 建议 + 直通车关键词 |
| AI 改图·拍摄简报 | **AI 四步改图流水线** + 5 张主图拍摄简报 |
| 图片诊断 | **图片评分 + 爆款套路挖掘 + 对比分析 + 改图优先级** |

**AI 四步改图流水线（SmartOptimizePipeline）：**
1. **找问题**：上传商品图 → AI 自动检测背景/色彩/构图/文案问题，标注严重程度
2. **看原因**：每个问题对销量的影响，引用同类目爆款对比数据（如"换白底图的商家点击率高 59%"）
3. **选方案**：勾选改进项（去背景/提亮色彩/裁切放大/加促销文案/标准比例），至少选 3 项
4. **出新图**：AI 生成 3 套优化方案（白底促销型/场景氛围型/细节展示型），带预估点击率和成交率
   - 选择方案 → 原图 vs 新图对比条 → 确认替换上线（暂停状态创建，安全可控）

**图片诊断（ScoringTab）：**
- 综合评分：0-100 环形图 + 优秀/及格/需改进判定
- 四维度评分条：构图/色彩/文案/版式
- 优劣势分析：做得好的地方 / 建议改进的地方
- 爆款图片套路：3 种拍图模式 + 投入产出比/点击率/成交率提升数据 + 样本数
- 你的图 vs 爆款图对比表：逐项对标，红色=需改/绿色=达标
- 改图优先级排序：按预期效果 × 成本排序（简单马上能改 / 需要时间 / 需要重拍）

**文案与 SKU：**
- 3 版标题（搜索/种草/活动导向）+ CTR 预估 + 渠道标注
- 详情页文案分屏结构：开场钩子（第一屏）→ 核心卖点（2-3屏）→ 场景描述（第4屏）→ FAQ（末屏）
- SKU 推荐：颜色上架顺序 + 预测最热组合
- 直通车关键词 5 个 + 建议出价 + 一键加入

**拍摄简报：**
- 5 张主图拍摄卡片（P0/P1/P2 优先级）
- 每张详细到：场景/光线/构图/道具/模特要求/图上文字/背景
- 状态追踪：待拍摄 → 拍摄中 → 已完成 → 测试中
- 一键复制拍摄单 + 参考图 + CTR 预估提升

### Skill 4：智能定价（运筹学动态定价引擎）

**三标签页工作台：**

| 标签页 | 功能 |
|--------|------|
| 核心结论 | 最优价 + 弹性系数 + 20 档价格表 + 利润曲线 |
| 阶段与曲线 | 新品期/成长期/大促期/清尾期自动切换 + Recharts 双轴折线图 |
| 风控护栏 | 负毛利硬拦截 / 深降价警告 / 日销下跌熔断 |

- **DID 因果推断**：剥离季节性和大促影响，得出真实价格弹性
- **LightGBM 冷启动预测**：新品无历史数据时的弹性估算
- **拉格朗日对偶优化**：在毛利约束下求 GMV 最大的价格
- **20 个价格档位枚举**：日销、GMV、月利润对比，最优行高亮
- **阶段化推荐**：新品期 → 成长期 → 大促期 → 清尾期自动切换

### Skill 5：评价诊断

**四标签页工作台：**

| 标签页 | 功能 |
|--------|------|
| 总览 | 评价总数 + 好评率 + 情感趋势 |
| 人群深潜 | **AI 生成买家画像** + 拟真评价 + 产品匹配度 |
| 评分维度 | LLM 情感分析（面料/版型/做工/色差/尺码）|
| 修复行动 | P0/P1/P2 问题 + 修复模板一键复制 |

**人群深潜（独特功能）：**
- AI 生成多个买家画像：姓名、头像、年龄、预算、核心诉求
- 每个画像有产品匹配度评分（0-10）
- 拟真评价叙事：模拟该画像买家的真实评价
- 强画像/弱画像分类，帮商家理解"谁在买你的货"

**跨 Skill 数据闭环：**
- 好评关键词自动回传 Skill 3 标题优化
- 差评痛点标签写入特征库，选款时自动过滤高风险属性

### Skill 6：推广诊断

**三标签页工作台：**

| 标签页 | 功能 |
|--------|------|
| 账户总览 | 健康分（0-100）+ ROI 对比基准线 + 每日浪费金额 |
| 关键词行动 | 暂停/加价/保持/新增 行动表 + 花费/订单/ROI |
| 预算与时段 | 最大 PPC 计算 + 投放高峰时段建议 + 优化后预估 ROI |

- **烧钱黑洞识别**：自动找出 ROI < 1 的关键词
- **最大 PPC 公式**：单价 × CVR × (1/目标ROI)
- **优化预估**：执行建议后的预期 ROI 提升

### Skill 7：活动促销（OR 驱动大促方案）

**四标签页工作台：**

| 标签页 | 功能 |
|--------|------|
| 核心方案 | 运筹学最优折扣 + 约束可视化 + 最优活动价 |
| 执行排期 | 预热/爆发/返场三阶段时间线 + 每阶段具体动作 |
| 情景与风险 | **三情景利润预测（P10/P50/P90 Monte Carlo 柱状图）** + 风险预警 |
| 保存收尾 | 保存到商品库 + 完整 JSON 导出 + 开始分析下一款 |

- **商品卡片列表**：健康分环形图 + Skills 完成进度条 + 关键指标预览

### 三层评估体系

## Slide 11：1.3 创意方案 — 技术架构与创新点

### 技术选型

| 层 | 选型 | 理由 |
|----|------|------|
| LLM | 通义千问 qwen-max | 淘宝生态、中文强 |
| 图像理解 | qwen-vl-max（多模态） | 图片→属性识别 |
| 图片生成 | GPT-image-1 / GPT-image-1-mini | OpenAI 最新图片编辑模型，支持蒙版 + 参考图 |
| 图片分割 | 浏览器端 ONNX Runtime | 零服务端成本，实时前景分割 |
| 向量检索 | FAISS CPU + OpenCLIP | 无需部署，本地跑 |
| 预测模型 | GATv2 + LightGBM | 已有验证结果 |
| 因果推断 | DID（双重差分） | 真实价格弹性 |
| 优化求解 | 拉格朗日对偶 + 枚举 | 数学最优定价 |
| 前端 | Next.js 16 + Tailwind CSS | 现代 SaaS 体验 |
| 状态管理 | Zustand | 跨 Skill 数据流转 |

### 六大核心创新

1. **真实 AI 选品模型**：GATv2 图神经网络 + LightGBM，非 LLM Prompt 套壳（S_F1=0.59）
2. **自研聚类引擎 三位一体**：风格聚类 + 趋势真假判断 + 爆款预测三层叠加
3. **运筹学动态定价**：运筹学 OR 模型，阶段化自动切换（新品/成长/大促/清尾）
4. **AI 实时图片生成闭环**：浏览器端 ONNX 分割 → 换色/图案预览 → GPT-image-1 生成 → 四步改图流水线 → 图片诊断评分
5. **买家画像深潜**：AI 生成多维度买家画像 + 拟真评价 + 产品匹配度评分
6. **不滥用 LLM**：结构化预测用小模型、优化用 OR 求解器、因果推断用统计方法、图片分割用 ONNX

### 模型选择策略

```
结构化预测 → GATv2/LightGBM（快、可解释）
约束优化   → OR 求解器（数学最优，有毛利保证）
向量检索   → FAISS + OpenCLIP embedding（高效无推理）
因果推断   → DID 统计方法（不被相关性误导）
前景分割   → 浏览器端 ONNX（零服务端成本）
图片生成   → GPT-image-1（蒙版编辑 + 参考图）
语言生成   → LLM（确实需要自然语言时才用）
图像理解   → 多模态 LLM（视觉推理）
```
